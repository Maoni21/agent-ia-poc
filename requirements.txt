# ================================================================================
# NOTES D'INSTALLATION
# ================================================================================

# INSTALLATION RECOMMANDÉE:
# 1. Créer un environnement virtuel:
#    python -m venv .venv
#    source .venv/bin/activate  # Linux/macOS
#    .venv\Scripts\activate     # Windows
#
# 2. Mettre à jour pip:
#    pip install --upgrade pip
#
# 3. Installer les dépendances:
#    pip install -r requirements.txt
#
# 4. Pour le développement:
#    pip install -r requirements-dev.txt
#
# 5. Pour l'API:
#    pip install -r requirements-api.txt
#
# DÉPENDANCES SYSTÈME REQUISES:
# - Python 3.10 ou supérieur
# - Nmap 7.80 ou supérieur
# - Git (pour certaines dépendances)
# - Compilateur C (pour certains packages comme cryptography)
#
# UBUNTU/DEBIAN:
#   sudo apt-get update
#   sudo apt-get install python3-dev python3-pip nmap git build-essential
#
# CENTOS/RHEL:
#   sudo yum install python3-devel python3-pip nmap git gcc
#
# MACOS:
#   brew install python nmap git
#
# WINDOWS:
#   - Installer Python depuis python.org
#   - Installer Nmap depuis nmap.org
#   - Installer Git depuis git-scm.com
#   - Installer Visual Studio Build Tools
#
# ================================================================================
# TROUBLESHOOTING - Résolution de problèmes courants
# ================================================================================

# PROBLÈMES COURANTS ET SOLUTIONS:
#
# 1. Erreur compilation cryptography:
#    - Ubuntu: sudo apt-get install libffi-dev libssl-dev
#    - CentOS: sudo yum install libffi-devel openssl-devel
#    - macOS: xcode-select --install
#
# 2. Erreur python-nmap:
#    - Vérifier que nmap est installé: nmap --version
#    - Permissions: certaines fonctions nécessitent sudo
#
# 3. Erreur OpenAI API:
#    - Vérifier la clé API dans .env
#    - Vérifier la connectivité Internet
#    - Vérifier les quotas API OpenAI
#
# 4. Erreur imports:
#    - Vérifier l'environnement virtuel activé
#    - Réinstaller: pip install --force-reinstall -r requirements.txt
#
# 5. Performance lente:
#    - Vérifier la version Python (3.10+ recommandé)
#    - Augmenter la mémoire disponible (4GB+ recommandé)
#
