## Glossaire

- **CVE** : identifiant public d’une vulnérabilité (ex: CVE-2021-44228).
- **CVSS** : score (0–10) représentant la sévérité technique.
- **NVD / NIST** : sources officielles (National Vulnerability Database).
- **Nmap** : outil de scan réseau.
- **NSE** : scripts Nmap (Nmap Scripting Engine).
- **Remédiation** : actions pour corriger/réduire le risque.
- **Rollback** : procédure de retour arrière en cas d’échec.
- **PoC** : preuve de concept (non production).
